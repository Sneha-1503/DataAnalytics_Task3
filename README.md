# Exploratory Data Analysis (EDA) Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail product dataset to uncover patterns, trends, and relationships among variables.

## Objective
Analyze the dataset using statistical summaries and visualizations to generate meaningful business insights.

## Dataset
The dataset includes:
- Product Information
- Brand
- Category
- Price
- Stock
- Availability
- Color
- Size

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## EDA Process
1. Data Understanding
2. Data Cleaning
3. Missing Value Analysis
4. Statistical Summary
5. Data Visualization
6. Correlation Analysis
7. Insight Generation

## Visualizations
- Product Category Distribution
- Availability Distribution
- Price Distribution
- Stock Distribution
- Brand Analysis
- Correlation Heatmap
- Scatter Plots

## Key Insights
- Identified dominant product categories.
- Analyzed inventory distribution.
- Explored pricing trends.
- Studied stock management patterns.
- Determined relationships between numerical features.
